# The Data

# Shared Database/API

Everyone on the Réseau du programme d'action uses the same database and API. All organizations hosting the components as an entire platform targeting specific types of commitments such as Earthmind, UNEP-WCMC and SCBD, referred to as network partners, are automatically and 100% interoperable between all other partner platforms.

# Schema.org

The data within the Réseau du programme d'action is modeled based on schema.org, the most popular vocabulary on the internet founded by Google, Microsoft, Yahoo and Yandex.

This allows our data to be indexed by the search engines in the format that they know and expect which can lead to increased ranks within the search engines and in turn more visibility with respect to outreach.

Furthermore the Réseau du programme d'action api is customizable for each partner via json-ld. Les partenaires can define their own classes and properties, adopt others from an external vocabulary or extend existing schema.org classes with their own properties. Similarly, the form component can be customized per partner to create and edit custom properties or custom wording for any property.

# Multilingual

The api is designed to take multiple language inputs per property. By default only returning one language when called. Similarly, the visual components will facilitate full translation.

# Integration of Existing Platform Data

With any set of systems, data can be interoperable in a number of ways. It appears this rarely happens. It sometimes takes resources from both parties to make the data available. Often a mapping system is created to convert one set to another. Then maintaining it, is a data source changes on either side the mapping function needs maintenance. The idea with the Réseau du programme d'action is we want to provide a free alternative to existing platforms looking to update their systems. When that time comes the free part is a big motivator to simply join the network and migrate their data. Once they join they benefit from the 100% interoperability among the partners.
